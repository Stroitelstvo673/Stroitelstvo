# СтройГрад — сайт строительной компании

Состав проекта:
- 15 HTML-страниц;
- адаптивная верстка;
- единый CSS-файл;
- JavaScript для мобильного меню, форм, FAQ и калькулятора стоимости;
- SEO meta description и keywords на каждой странице.

Запуск:
Откройте файл `index.html` в браузере. Для проверки физической структуры сайта достаточно сохранить папку проекта целиком.
